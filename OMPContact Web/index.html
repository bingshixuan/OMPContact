Outer Membrane Protein Target Binding Site Prediction

OMPContact Prediction is an automatic server for predicting the binding site between amino acids.

User Mannal

1. Prepare your test samples. Test samples should be organized as fasta file format. It should be mentioned that, the first line of fasta file, wich contain some information about amino acids seems not relate to the compute must input into the blank under step 1.

2. Choose covariance method. Distance metric computed by covariance. OMPContact use three different covariance to build the model. Each covariance method has built a different model. Choose one model before you want to get the contact result.

3. Get predict result. Computing procedure cost some times. OMPContact web server will send you a E-mail after finishing the computing. Make sure correct E-mail address has been filled up the textfield before press the submit button.

4. Result files.There are two files attached to your received E-mail. The one which is end with "line" provide you a sorted contact ratio of each amino acid to another. The other file will give you a whole sight by upper triangle of square matrix of amino acid sequence contact percentage. The row and line tittle are list of amino acids in sequence.
