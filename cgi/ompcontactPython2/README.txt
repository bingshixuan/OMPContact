1. Usage : python OMPContact.py -i input.fasta -m {ELSC | MI | OMES} -o output.pred

2. Installation
	(1) Betaware environment configuation, BETAWARE_ROOT=.../betaware
	(2) mview lib configuation, please reference https://github.com/desmid/mview
	(3) Java Installation. Basically, Unix system include Java Program. If you want to download Java Program, Please contact http://www.oracle.com/
	(4) libsvm Installation. If you computer do not hava libsvm library, This program will use default libsvm. Packages will compiled by this program
